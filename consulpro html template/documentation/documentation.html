Change Typography

To change typography go to "app/scss/style.scss" file. Change the google font import url. And also need to edit "_typography.scss" file.

app/scss/style.scss 
                                            
@import url('https://fonts.googleapis.com/css2?family=Fira+Sans:wght@500;600;700&family=Poppins:wght@400;500;700&display=swap');

Also need to "_variable.scss" edit to change font family.

app/scss/_typography.scss 

body {
    font-family: $font;
    background-color: $brand-white;
    color: $color;
    font-weight: 400;
    font-size: 16px;
    line-height: 25px;
    position: relative;
    height: 100%;
    box-sizing: border-box;
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
}

        background: url('../images/home3-graphics.png')no-repeat left bottom / contain;
        z-index: -1;
    }

Change Style


We are use sass, gulp, browsersync, node js work flow for this template.

It will help you to customize the html, css/scss, js files smothly and fastly. If you know sass/scss gulp code please open the "consulpro" folder on your favourite editor. Go to terminal type npm install. It will insall all packages via package.json file. After installing all pages. Please write "gulp". This gulp command will convert scss files to css. All is located on "app" folder. I've added / imported all scss file via style.scss. If you want to customize you can do here.

If you not scss code customize then you will fiend style.css file inside "css" folder. You can change here. I've commented all sections css via index commenting. You can find it easily.

Change Header

Consulpro have 4 header variation.To change the header, you have to replace the current header with the header that you want. For example:

index.html 
                                            
 <!-- Index Header Start -->
<header class="header index-header">
    ....
    ....
</header>
<!-- Header End -->

Lets say you want to replace the current header with the header 2. Open file 'header2.html' and find the following code:

header-2.html 
                                            
<!-- Header 2 start -->
<header class="header header-2">
    ....
    ....
</header>
<!-- Header End -->

Copy everything from the 'Header 3 Start' till the 'Header End' with everything that is in it and replace with the whole code of your current header (as shown on the previous example).

header-3.html 
                                            
<!--  Header 3 Start -->
<header class="header header-3">
    ....
    ....
</header>
<!-- Header End -->

Copy everything from the 'Header Start' till the 'Header End' with everything that is in it and replace with the whole code of your current header (as shown on the previous example).

All Inner page Header 
                                            
 <!-- Header Start -->
 <header class="header header-bg">
    ....
    ....
</header>
<!-- Header End -->

Contact Form

Our Contact form is Ajax and php based. Contact form will use php to send emails. So open mailer.php file that can be found on the php folder. Change the current email "metariqul91@consulpro.com" to your email address and you are ready to go.


NOTE: Contact form require web server to send emails. Emails can go to your spam/junk folder.

mailer.php 
                                            
# FIX: Replace this email with recipient email
$mail_to = "metariqul91@gmail.com";

Google Map

To change the location that is shown on the maps you have to replace the langtitude and longtitude.


First you have to find the langtitude and longtitude of the location that you want to be shown on the google maps. For that you have to go to Google Maps and pin out your location. Below the address of your location the langtitude and longtitude will be shown as on the image below:


After copying the langtitude and longtitude you will open custom.js file that can be found on the JS folder.

js/google-map.js 
                                            
//Map Js
var wWidth = $(window).width();
var googleMapSelector = $('#google-map'),
myCenter = new google.maps.LatLng(40.7348213, -73.7744253);
